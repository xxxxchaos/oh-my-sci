# omo-sci 快速开始

## 启动项目

在项目目录中，启动 omo-sci 项目向导：

```
/sci-start
```

Dubin（主编排 agent）会与你对话，收集以下信息：

1. 研究主题和目标
2. PICO 问题（如适用）
3. 研究方法偏好
4. 目标期刊
5. 数据情况

## 完成意图访谈（阶段 0）

Dubin 会引导你完成结构化访谈，确保所有关键信息都被覆盖。访谈完成后自动生成研究蓝图。

## 研究设计（阶段 1）

Archimedes agent 会根据蓝图设计研究方案，包括：

- 研究类型（RCT、队列研究、病例对照等）
- 样本量估算
- 统计分析计划
- 执行计划

## 数据分析（阶段 2）

SPSSer agent 会执行统计分析，包括：

- 描述性统计
- 推断性统计
- 敏感性分析
- 可视化图表

## 论文撰写（阶段 3）

Writer agent 会根据分析结果撰写论文初稿，Polisher agent 进行语言和逻辑润色。

## 投稿准备（阶段 4）

Submitter agent 帮助你准备投稿所需的所有材料。

## 完整性检查

流水线包含两道完整性闸门：

- **闸门 I**（分析完成后）：检查主张与证据的一致性
- **闸门 II**（终稿完成后）：零容忍终审

## 全部指令

| 指令 | 功能 |
|------|------|
| `/sci-start` | 启动新项目 |
| `/sci-doctor` | 环境诊断 |
| `/sci-status` | 项目状态查询 |
| `/sci-usage` | 用量和配额查询 |

## 首次使用建议

1. 运行 `/sci-doctor` 确认环境就绪
2. 配置模型（见[模型配置指南](./model-setup.md)）
3. 运行 `/sci-start` 开始第一个项目
4. 跟着 Dubin 的引导走完全程
